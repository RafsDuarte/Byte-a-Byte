# 💻 Byte A Byte — Django

Site educacional com dicas sobre hardware, peças de computador e montagem de PCs, feito em Django e Python.

---

## 📌 Sobre o Projeto

O **Byte A Byte** é um blog voltado para aprendizado sobre informática e hardware. O objetivo é publicar conteúdo sobre:

- Processadores (CPU)
- Placas de vídeo (GPU)
- Memória RAM
- Armazenamento (SSD, HDD, NVMe)
- Placas-mãe
- Fonte de alimentação
- Coolers e refrigeração
- Montagem e manutenção de PCs
